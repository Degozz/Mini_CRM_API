## ЗАПУСК ПРОЕКТА
### 1. Подготовка окружения
В корневом каталоге проекта переименуйте файл-шаблон с переменными окружения .env.example на .env:
Подставьте в этот файл свои данные

### 2. Запуск контейнеров
```bash
docker compose up --build
```

Docker поднимет базу данных, создаст структуру таблиц из init.sql, скомпилирует Go-код и запустит веб-сервер на порту 8080.

## ПРИМЕРЫ API-ЗАПРОСОВ

### 1. Проверка работоспособности (Health Check)
* **Запрос:** `GET http://localhost:8080/health`
* **Ответ (200 OK):** `"OK"`

### 2. Создание клиента
* **Запрос:** `POST http://localhost:8080/clients`
* **Тело запроса (JSON):**
```json
{
  "name": "ООО Ромашка",
  "email": "client@example.com",
  "phone": "79990000000",
  "status": "new"
}
```

### 3. Создание заявки для конкретного клиента
* **Запрос:** `POST http://localhost:8080/clients/1/leads`
* **Тело запроса (JSON):**
```json
{
  "title": "Подключение метода оплаты",
  "description": "Клиент хочет подключить новый метод",
  "status": "new"
}
```
### 4. Получение списка клиентов (с фильтрацией)
* **Запрос:** `GET http://localhost:8080/clients?status=new&email=client@example.com`

### 5. Получение клиента по ID
* **Запрос:** `GET http://localhost:8080/clients/1`

### 6. Получение списка заявок клиента
* **Запрос:** `GET http://localhost:8080/clients/1/leads`
